# Algoritmo para detectar pessoas

Este é um algoritmo para segurança residencial.

Ele salva as imagens das pessoas em pastas com o titúlo sendo a hora atual.

## Como executar?
É necessário ter o python instalado.

terminal linux:
> python2 main.py
